# BeadNeko v1.0 功能清单（MVP）

## 核心功能
- [ ] 图片上传（相册 / 拍照）
- [ ] 图片像素化（可调像素尺寸：16×16 到 128×128）
- [ ] 拼豆网格渲染（方形点、网格线、缩放）
- [ ] 基础色卡映射（内置 20–30 色基础色）
- [ ] 自动统计颜色使用数量
- [ ] 生成拼豆像素预览图
- [ ] 导出 PNG（保存到相册）

## 界面页面
- [ ] 首页（上传图片、查看历史入口）
- [ ] 图片选择页
- [ ] 参数调整页（像素尺寸、颜色数量）
- [ ] 拼豆预览页（可放大查看）
- [ ] 导出页（保存像素图到本地）
- [ ] 历史记录页（缩略图 + 再次打开）
- [ ] 设置页（隐私政策、版本信息）

## 本地存储
- [ ] 本地保存作品缩略图与参数
- [ ] 历史记录（Hive）

## 权限与系统支持
- [ ] 相册权限（NSPhotoLibraryUsageDescription）
- [ ] 相机权限（NSCameraUsageDescription）
- [ ] Android 文件权限

## UI 风格
可爱粉色/蓝绿渐变（亲和）
图标：猫脸 + 拼豆点
按钮：圆角、暖色
尽量简洁，第一版以工具化为主
