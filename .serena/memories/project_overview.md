# BeadNeko Flutter App Overview

- Purpose: BeadNeko is a Flutter app for bead art (perler bead) enthusiasts. Users import a photo, convert it to bead-style pixel art, view a grid with color mapping, and export/save results.
- MVP features are outlined in `docs/FEATURES.md` (image upload, pixel sizing, grid rendering, color mapping/counts, export, history, settings).
- Platform: Flutter mobile app (Android/iOS).
- Entry point: `lib/main.dart`.
