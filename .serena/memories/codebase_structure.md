# Codebase Structure

- `lib/main.dart`: app entry point
- `lib/components`: reusable UI components
- `lib/pages`: screens/pages
- `lib/models`: data models
- `lib/core`: core logic/services
- `lib/utils`: helpers/utilities
- `lib/i18n`: localization resources
- `lib/constants.dart`, `lib/enums.dart`, `lib/theme.dart`, `lib/store.dart`: shared constants, enums, theming, state/store
- `android/`, `ios/`: platform-specific projects
- `docs/FEATURES.md`: MVP feature checklist and UI guidance
- `scripts/`: project setup and utility scripts
