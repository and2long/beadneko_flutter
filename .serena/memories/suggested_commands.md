# Suggested Commands (Darwin)

## Setup
- `flutter pub get`

## Run
- `flutter run`
- `flutter run -d <device_id>` (list devices via `flutter devices`)

## Test
- `flutter test`

## Lint/Analyze
- `flutter analyze`

## Format
- `dart format lib test`

## Scripts
- `bash scripts/init.sh` (initial project customization)
- `bash scripts/uninstall_android_app.sh`
- `bash scripts/uninstall_ios_app.sh`

## Common utils (Darwin)
- `ls`, `cd`, `pwd`, `rg`, `find`, `git status`, `git diff`
