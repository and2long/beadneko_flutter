# Tech Stack

- Flutter (Dart SDK ^3.10.1)
- State management: provider
- Networking: dio
- Storage: shared_preferences, path_provider
- Image handling: image_picker, image, image_gallery_saver_plus
- UI helpers: flutter_smart_dialog, cached_network_image, flutter_spinkit, google_fonts
- Permissions/OS integration: permission_handler, share_plus, url_launcher, package_info_plus
- Localization: flutter_localizations
- Linting: flutter_lints via `analysis_options.yaml`
